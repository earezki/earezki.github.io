# Dev|Journal

## Quick Start

1. Install dependencies
2. Start the dev server

```bash
npm install
npm run dev
```

Open http://localhost:4321

## Content

posts live in `src/content/posts`. Each has frontmatter:

```yaml
title: "Post title"
pubDate: 2024-09-09T00:00:00.000Z
categories: [Category]
slug: custom-slug
```

## RSS & Sitemap

RSS feed: `/rss.xml`  
Sitemap: `/sitemap-index.xml` (auto-generated)
